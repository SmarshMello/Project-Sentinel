# Project Sentinel 5.1 Desktop Handoff

## Recommended package
Use `Project-Sentinel-5.1-CHANGED-FILES-ONLY.zip`.

## Installation
1. Open the repository in GitHub Desktop.
2. Choose **Repository > Show in Explorer**.
3. Extract the changed-files ZIP.
4. Copy its contents into the repository root.
5. Allow Windows to merge folders and replace matching files.
6. Commit: `Project Sentinel 5.1 - Connected Intelligence`
7. Push origin and wait for GitHub Pages.
8. Hard refresh with Ctrl+F5.
