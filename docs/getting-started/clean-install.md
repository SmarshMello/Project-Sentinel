---
title: Clean GTA V Legacy Installation
---

## Goal
Start with a verified, unmodified GTA V Legacy installation.

## Steps
1. Install GTA V Legacy through the official launcher.
2. Launch Story Mode once and close it.
3. Right-click `GTA5.exe` → **Properties → Details** and record the Product Version.
4. Create a separate **Project Sentinel Mod Vault** outside the GTA folder.
5. Keep the modded installation out of GTA Online.

## Test gate
Story Mode launches without RAGE Plugin Hook, Script Hook V or any mods.
