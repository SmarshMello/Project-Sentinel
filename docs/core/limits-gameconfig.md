## Installation
Install Heap Adjuster and Packfile Limit Adjuster in the GTA root. Replace `gameconfig.xml` only in the mods copy of `update.rpf`, using a configuration matching the exact game build.

## Files and placement
Read the creator's included documentation and preserve the package's folder structure. Do not replace newer shared dependencies with older bundled copies without checking versions.

## Test gate
Story Mode and LSPDFR load without allocation or packfile errors.

## Common mistake
Installing several unknown plugins at once makes failures difficult to isolate.
