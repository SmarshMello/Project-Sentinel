## Installation
Use the manual **GTA V Legacy** package and merge it into the GTA root. Launch through RAGE Plugin Hook.

## Files and placement
Read the creator's included documentation and preserve the package's folder structure. Do not replace newer shared dependencies with older bundled copies without checking versions.

## Test gate
Create an officer, go on duty and complete a basic traffic stop.

## Common mistake
Installing several unknown plugins at once makes failures difficult to isolate.
