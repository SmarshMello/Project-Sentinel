## Installation
Install OpenIV, select Windows → Grand Theft Auto V, point it to the Legacy root, then use **Tools → ASI Manager** to install ASI Loader and OpenIV.asi.

## Files and placement
Read the creator's included documentation and preserve the package's folder structure. Do not replace newer shared dependencies with older bundled copies without checking versions.

## Test gate
OpenIV browses the archives and the safe mods folder appears purple.

## Common mistake
Installing several unknown plugins at once makes failures difficult to isolate.
