# Sentinel Watcher 0.7 — Daily Intelligence Update

## Included

- Daily automatic scans through GitHub Actions (`09:23 UTC` every day).
- Per-project risk level: low, medium, or high.
- Confidence score based on source type and scan quality.
- Action recommendation for every monitored project.
- Category-level ecosystem health summaries.
- New dashboard intelligence cards and risk indicators.
- Report schema 6 and Watcher version 0.7.0.

## Safety rule

Watcher intelligence is advisory. It does not automatically mark a mod compatible, incompatible, dead, or safe to install. High-impact decisions still enter the human review queue.
